Connection Logic

  • • Client message → both Per-message classifier and Conversation watcher
  • • External signal → Signal monitor
  • • Trip / state change → Scheduled query
  • • Ops prompts agent → Agent mode (skips review, splits into AI + human sub-tasks — annotate this)
  • • Ops spots → bypasses AI lane entirely, goes direct to "Manual create" → "Task created"
  • • Per-message classifier auto tier → direct to "Task created"
  • • Per-message classifier assist/escalate, Conversation watcher, Signal monitor, Scheduled query → all flow into the human review gateway
  • • Review approve/edit → "Task created"
  • • Review reject → "Dismissed"
